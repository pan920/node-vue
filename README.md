# vue+node+mongodb全栈项目
## 适合初学node的朋友撸着玩
+ 目前功能: 1.登陆、注册、包括部分角色与路由的关系
          2.列表的增删该查
          3.建议查询统计
+ 项目启动
~~~
    启动vue（默认client目录）和node: npm run dev
~~~
~~~
    老的ui(简易搭建)
    单独启动vue cd到client
    npm run serve
~~~
~~~
    单独启动node
    npm run server
~~~
~~~
    新增ui(vue-element-admin)
    cd 到 client-admin
    执行 npm install
         npm run dev
    打包 npm run build:stage
~~~
